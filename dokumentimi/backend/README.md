# Dokumentimi Detajuar: Backend

Ky direktorium përmban dokumentacion të detajuar për backend-in (FastAPI). Pikat që duhen dokumentuar më hollësisht:

- Arkitektura e modules (routers, services, models, schemas)
- Rrjedhat e autentifikimit dhe autorizimit
- Integrimet (Supabase, Redis)
- Mënyra e menaxhimit të skedarëve dhe storage
- Testimi (unit/integration) dhe fixtures
- Mënyra e deployimit të backend-it

Referenca fillestare: `03-backend-implementimi.md`

Shtoni nën-skeda si: `routes.md`, `services.md`, `tests.md`, `deploy.md` sipas nevojës.
