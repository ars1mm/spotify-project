# Dokumentimi Detajuar: Frontend

Ky direktorium përmban dokumentacion të detajuar për frontend-in (Next.js + TypeScript).

Tema për t'u përfshirë:

- Strukturë komponentesh dhe kontekste (`PlayerContext`, `AuthContext`)
- Rrjedhat e autentifikimit dhe token management
- Integrimi me backend (API client, caching)
- Player audio dhe menaxhimi i queue
- Testing i komponentëve dhe E2E
- Best practices për performance dhe accessibility

Referenca fillestare: `02-frontend-implementimi.md`

Shtoni nën-skeda si: `components.md`, `contexts.md`, `testing.md`, `build.md`.
