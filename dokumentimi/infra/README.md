# Dokumentimi Detajuar: Infrastrukturë (Kubernetes, Monitoring, Secrets)

Ky direktorium duhet të centralizojë udhëzimet për deployment në infrastrukturë, monitoring dhe best practices:

- Kubernetes manifests dhe shembuj (referencë: `backend/k8s/`)
- Secrets management (Kubernetes Secrets, Vault)
- Monitoring & metrics (Prometheus, Grafana)
- Logging centralized (ELK/Graylog/etc.)
- Autoscaling, health checks, readiness/liveness probes

Shtoni nën-skeda si: `k8s.md`, `monitoring.md`, `secrets.md`, `logging.md`.
